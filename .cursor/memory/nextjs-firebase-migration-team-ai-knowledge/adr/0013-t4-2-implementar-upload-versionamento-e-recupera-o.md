# ADR 0013: T4.2 Implementar upload, versionamento e recuperação de arquivos

- Status: Accepted
- Data: 2026-05-28

## Contexto
Implementação da tarefa T4.2 do plano de migração Next.js + Firebase.

## Consequências
A base da nova aplicação `apps/frontend` está estruturada e integrada aos fluxos MCP por workspace.
